# 🎮 AI游戏创作助手

集成Aippy式AI游戏创作能力，让用户通过自然语言描述快速生成可玩小游戏，支持竖屏即刷即玩、双栏实时编辑、Remix二创，让每个人都能零门槛创作游戏。

## 核心功能

### 🎯 AI自然语言生成
用户通过自然语言描述游戏需求，AI会自动生成可玩的互动小游戏或数字玩具。

示例Prompt：
- "做一个打僵尸的小游戏"
- "设计一个坦克闯关游戏"
- "创建一个跳跃收集金币的游戏"

### 📱 竖屏Feed流展示
采用类似TikTok的竖屏信息流形式，用户向上滑动即可即点即玩，无需下载安装或等待加载。

### 🔄 双栏实时编辑
创作界面提供Chat和Preview两个栏目：
- **Chat栏**：用自然语言修改功能或修复问题
- **Preview栏**：实时预览游戏效果

示例：
- "增加连击功能"
- "加入血量条显示"
- "让敌人移动速度快一点"

### ✂️ Remix二创机制
支持一键改编社区内的任何作品，用户可基于现有游戏：
- 添加新功能
- 调整参数
- 更换主题
- 形成内容持续生长的创作链条

### 💡 Idea灵感模板
内置创意框架库，为缺乏灵感的新手提供预设模板：
- 跳跃类游戏模板
- 射击类游戏模板
- 益智类游戏模板
- 角色扮演模板

## 使用方法

### 基础创作流程

1. **启动创作模式**
   ```bash
   molili ai-game-creator create
   ```

2. **描述游戏需求**
   在Chat栏输入自然语言描述，例如：
   ```
   做一个打僵尸的小游戏，玩家控制一个角色，用武器攻击从右侧过来的僵尸
   ```

3. **查看生成结果**
   Preview栏会实时显示生成的游戏，可直接试玩

4. **迭代优化**
   在Chat栏继续提出修改需求：
   ```
   增加得分系统，每打中一个僵尸得10分
   ```

5. **导出游戏**
   完成后导出为HTML5格式，可分享到任何平台

### Remix二创流程

1. **选择要Remix的游戏**
   ```bash
   molili ai-game-creator remix <game-id>
   ```

2. **提出修改想法**
   ```
   把这个游戏改成科幻主题，把僵尸改成机器人，背景改成太空
   ```

3. **生成新版本**
   系统会基于原游戏生成新版本，保留原有逻辑

4. **发布分享**
   可以将Remix作品发布到社区

### 使用灵感模板

```bash
molili ai-game-creator templates
```

列出所有可用模板，然后选择一个开始创作：
```bash
molili ai-game-creator create --template jumping-game
```

- 导出的HTML文件可以在本地运行，也可以部署到服务器

## 示例对话

用户：做个打僵尸游戏
AI：好的，我来为你创建一个打僵尸小游戏。玩家控制左侧角色，用键盘空格键发射子弹攻击从右侧来的僵尸，随着时间推移僵尸数量会增加。

用户：增加连击功能
AI：已添加连击系统！当你在2秒内连续打中3个或更多僵尸时，会触发连击状态，显示连击数，每连击一个额外获得5分。

用户：加入血量条显示
AI：血量条已添加！屏幕上方显示玩家血量（红色条），被僵尸碰到会扣血，血量归零游戏结束。同时添加了简单的生命回复道具。
