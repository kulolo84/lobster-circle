# 🎮 AI游戏创作助手

集成Aippy式AI游戏创作能力，让每个人都能零门槛创作游戏。

## 快速开始

### 安装

```bash
# 将技能克隆到OpenClaw技能目录
git clone https://github.com/kulolo84/ai-game-creator
```

### 使用

```bash
# 启动创作模式
molili ai-game-creator create

# 查看模板
molili ai-game-creator templates

# Remix现有游戏
molili ai-game-creator remix <game-id>
```

## 主要功能

✅ **一句话生成游戏** - 用自然语言描述想法，AI自动创建可玩游戏
✅ **竖屏即刷即玩** - 类似TikTok的竖屏体验，无需下载
✅ **双栏实时编辑** - Chat修改+Preview预览，即时生效
✅ **Remix二创** - 一键改编他人作品，创意生长
✅ **灵感模板** - 内置创意框架，快速上手

## 示例

**创建游戏：**
```
输入：做一个打僵尸的小游戏
输出：可玩的僵尸射击游戏，角色在左，僵尸从右来
```

**优化游戏：**
```
输入：增加连击功能
输出：添加连击系统，2秒内连续击中3个以上触发连击奖励
```

**Remix游戏：**
```
输入：把这个游戏改成科幻主题，僵尸改成机器人
输出：科幻版僵尸游戏，视觉和音效全部更新
```
